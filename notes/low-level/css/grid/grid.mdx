# grid

```css
display: "grid";
grid-template-columns: "1fr 1fr 1fr";
grid-template-rows: "1fr 1fr";
```

can rearrange order

- Grid Container. Direct parent of all the grid items.
- Grid Item. `Div`s (usually) Direct descendants of the grid container. Not the sub descendents.
