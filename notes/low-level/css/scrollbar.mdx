# Scrollbar

```css
* {
  scrollbar-color: #222e4d #08062b;
  scrollbar-width: thin;
  scrollbar-gutter: auto;
}

*:hover {
  scrollbar-color: #bcc8e2 #08062b;
  scrollbar-width: thin;
  scrollbar-gutter: auto;
}
```

Works and looks great everywhere, even Safari

```css
html {
  color-scheme: light dark;
}
```
